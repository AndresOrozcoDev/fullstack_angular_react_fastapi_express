## Ejecucion de desarrollo

Instalacion de dependencias
```bash
npm i
```

Crear el servidor
```bash
ng serve
```

## Andamio de codigo

Despliegue
```bash
ng build
```

Pruebas unitarias
```bash
ng test
```
