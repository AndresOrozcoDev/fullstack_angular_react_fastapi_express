## Ejecucion de desarrollo

Instalacion de dependencias
```bash
npm i
```

Crear el servidor
```bash
npm start
```

## Andamio de codigo

Despliegue
```bash
npm run build
```

Pruebas unitarias
```bash
npm test
```
